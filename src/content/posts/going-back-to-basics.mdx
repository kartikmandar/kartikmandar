I feel like nowadays with the advent of tools like Claude Code, Codex, and Cursor, coding in general has become quite reachable for a lot of people. Also, AI in general has been helping people in all sorts of workflows, and that's amazing. I personally have been an active advocate for using AI. But now I want to wind down a little. I will still be using AI a lot, but at the same time focus on active, non-AI-assisted learning.

I feel there is something about writing your equations and code by hand that fires the neurons, which isn't the same as guiding an AI and writing prompts. But at the same time, I need to keep up with all the new stuff that is happening and keep on using AI, of course.

It's an amazing time to be in.
Almost like seeing the future come to you in real time.
